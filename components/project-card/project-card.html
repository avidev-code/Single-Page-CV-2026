Problem: I redesigned LinkedIn's news feed to address inefficient navigation and excessive white space, which negatively impacted the user experience.

Process: I conducted an initial UX audit of the current feed, identified key user pain points, and applied UX Laws to propose strategic design solutions. I developed a mini Design System and a high-fidelity prototype.

Solution: I delivered a visually appealing and cohesive news feed redesign that optimizes space and improves navigation. The solution focused on a more logical organization and a consolidated layout, resulting in a more fluid and functional user experience.
